# Настройка RTS сцены и камеры

## Обзор
Созданы системы управления камерой и границами карты для RTS проекта.

## Созданные компоненты

### 1. MapBounds.cs
Управляет границами карты (в данный момент поддерживает круглую карту).

**Параметры:**
- `mapRadius` - радиус карты (по умолчанию 50)
- `mapCenter` - центр карты (по умолчанию 0,0,0)
- `isCircularMap` - использовать круглую карту (включено)
- `showBoundsGizmo` - показывать границы в редакторе (включено)
- `boundsGizmoColor` - цвет отображения границ

**Методы:**
- `IsWithinBounds(position)` - проверить в ли позиция в границах
- `ClampToBounds(position)` - вернуть позицию в пределах границ

### 2. RTSCameraController.cs
Контролирует камеру для RTS игры.

**Параметры движения:**
- `moveSpeedCurve` - кривая скорости движения камеры в зависимости от зума (EaseInOut от 40 до 10)
- `enableEdgePanning` - панорамирование с краев экрана (включено)
- `edgePanThreshold` - расстояние от края для панорамирования (20px)

**Параметры зума:**
- `minZoom` - минимальный зум (20)
- `maxZoom` - максимальный зум (80)
- `zoomSpeed` - скорость приближения к целевому зуму (10)
- `scrollZoomSpeed` - скорость зума колесом мыши (5)

**Управление:**
- **WASD** - движение камеры
- **Средняя кнопка мыши (скролл)** - перетаскивание камеры
- **Края экрана** - автоматическое панорамирование
- **Скролл мыши** - зум

### 3. MapBoundsVisualizer.cs
Визуализирует границы карты как элемент сцены (опционально).

## Инструкции по настройке

### Шаг 1: Подготовка сцены
1. Откройте сцену `Assets/Scenes/SampleScene.unity`
2. Удалите дефолтный куб и другие объекты (если они есть)

### Шаг 2: Создание MapBounds
1. В иерархии создайте новый пустой GameObject: `Right Click > Create Empty`
2. Переименуйте его в `MapManager`
3. Добавьте компонент `MapBounds` (Add Component > MapBounds)
4. Настройте параметры:
   - Map Radius: 50
   - Is Circular Map: включено
   - Show Bounds Gizmo: включено (для видимости в редакторе)

### Шаг 3: Настройка камеры
1. Выберите главную камеру (Main Camera)
2. Установите:
   - Position: (0, 30, 0)
   - Rotation: (90, 0, 0) - для top-down вида
   - Projection: Orthographic (важно для RTS!)
   - Orthographic Size: 30
3. Добавьте компонент `RTSCameraController` (Add Component > RTSCameraController)
4. Настройте параметры по необходимости (или используйте дефолтные)

### Шаг 4: Визуализация границ (опционально)
1. Создайте новый пустой GameObject: `MapBoundaryVisualizer`
2. Добавьте компонент `MapBoundsVisualizer`
3. Создайте простой материал для границ (белый или зеленый)
4. Назначьте материал в параметре `Boundary Material`

### Шаг 5: Освещение
1. В сцене должен быть главный свет (Main Light)
2. Настройте его для космической атмосферы:
   - Type: Directional
   - Color: светло-голубой или белый
   - Intensity: 0.7-1.0

## Проверка работы

1. Запустите игру (Play)
2. Используйте WASD или краевое панорамирование для движения камеры
3. Прокрутите колесо мыши для зума
4. Проверьте, что камера не выходит за границы карты
5. Должны быть видны границы (зеленый круг в редакторе)

## Советы

- Отрегулируйте `moveSpeedCurve` в зависимости от размера карты и предпочитаемой скорости (кривая скорости от зума)
- Используйте `edgePanThreshold` для удобного панорамирования (меньше = чувствительнее)
- Для большой карты может понадобиться увеличить `maxZoom`
- Границы карты автоматически ограничивают движение камеры
